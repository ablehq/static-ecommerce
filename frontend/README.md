# Steps for setting up eleventy

```
yarn init
yarn install
```

- copy gitignore file from https://www.gitignore.io/api/node
- initialize git repo
- initial git commit

## Add Eleventy

```
yarn add @11ty/eleventy
npx @11ty/eleventy --serve >> server
npx @11ty/eleventy >> build
```

Create file: _index.html_
